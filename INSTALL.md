# Как установить приложение Lavmee на телефон

## Для Android

### Способ 1: Chrome
1. Откройте сайт **lavmee.ru** (или **futuredate1.vercel.app** если первый не работает)
2. Нажмите три точки в правом верхнем углу
3. Выберите **"Добавить на главный экран"** или **"Установить приложение"**
4. Нажмите **"Установить"**
5. Приложение появится на экране телефона

### Способ 2: Samsung Internet
1. Откройте сайт в браузере Samsung
2. Нажмите три линии → **"Добавить страницу в"** → **"Главный экран"**

### Способ 3: Яндекс Браузер
1. Откройте сайт
2. Нажмите шестерёнку → **"На главный экран"**

---

## Для iPhone (Safari)

1. Откройте сайт **lavmee.ru** (или **futuredate1.vercel.app**)
2. Нажмите кнопку **"Поделиться"** (квадрат со стрелкой вниз)
3. Прокрутите вниз и выберите **"На экран Домой"**
4. Нажмите **"Добавить"**
5. Приложение появится на рабочем столе

---

## Если кнопки установки нет

- Убедитесь что сайт открывается (может потребоваться VPN если lavmee.ru не работает)
- Попробуйте другой браузер
- Используйте резервный адрес: **https://futuredate1.vercel.app**

---

После установки приложение будет работать как обычная программа — без адресной строки браузера.
