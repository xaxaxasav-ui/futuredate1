# **App Name**: FutureDate AI

## Core Features:

- Dynamic AI Personality Assessment: Interactive web interface for users to engage with an AI conversational tool that conducts dynamic personality tests and generates a detailed personality profile and vector, stored using Supabase.
- AI-Powered Matchmaking: An intelligent system that analyzes AI-generated personality vectors (stored in Supabase with pgvector) to suggest highly compatible user matches through a web dashboard, moving beyond superficial preferences.
- Secure User Accounts & Profiles: A comprehensive web-based user registration and login system with secure authentication (Email/Password, social logins) and profile management, backed by Supabase Auth and its PostgreSQL database.
- Real-time Messaging: An integrated web chat feature enabling matched users to communicate instantly and in real-time via text messages using Supabase Realtime before arranging a virtual date.
- Immersive Virtual Video Dates: A web interface providing functionality for secure video calls between matched users for virtual dates, potentially enhanced with an AI tool offering conversational icebreakers based on shared interests.
- Automated Content Moderation: An AI-driven system that automatically filters inappropriate user-submitted content and photos, including nudity detection, to ensure a safe and respectful platform environment.

## Style Guidelines:

- The visual scheme embraces a dark theme, evoking a sense of cyber-futurism and cosmic allure. The primary interactive color is a vibrant electric blue (#42BFFF), which pops against the dark backdrop. This hue reflects innovation and dynamism.
- The background features a deep, subtly blue-tinged charcoal (#161A1C), providing a sleek, expansive canvas. Its low saturation maintains a minimalist, unobtrusive feel while harmonizing with the primary color.
- A striking neon cyan (#07CCEC) serves as the accent color. This bright and highly saturated shade provides essential visual contrast for critical actions and highlighted elements, reminiscent of futuristic holographic displays.
- Headlines utilize 'Space Grotesk' (sans-serif), lending a computerized, techy, and modern feel suitable for a 'Future Dating' application. Body text employs 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, perfect for AI conversational elements.
- An ultra-minimalistic 'glassmorphism' inspired interface design, focusing on clean lines, ample negative space, and translucent elements to create a sophisticated, next-generation user experience.
- Subtle yet fluid transitions between screens and interactive elements. Special attention given to 'AI thought' animations during personality assessment to visually represent the intelligence at play, creating an engaging user journey.
- Modern, minimalist, and geometric icon designs. Icons will have a slight futuristic flair, using clean strokes and occasional neon glow effects to align with the cyberpunk aesthetic without being overly complex.
